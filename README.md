# Pixel Art Renderer

## Description

Renders pixel art across various views, including the console/terminal and React web pages components.

## Features

- Create and group pixels into Layers and a Canvas
- Renders pixels to the various views, including:
    - Terminal in black and white
    - Terminal in color
    - Terminal in an animated scene
    - React-driven web output
    - Basic text file
- Stores and saves data for import/export
